Current task
- Build a Java-only (no XML layout) file Find/Replace tool with:
  - Open file
  - Whole-word search
  - Regex/pattern search
  - Replace / Replace All
  - Save / Save As

How to use (device)
- Open the app drawer entry: "Find/Replace"
- Tap OPEN → choose a text file
- Enter Search (+ optional Whole word / Regex)
- Use FIND NEXT / REPLACE / REPLACE ALL
- Use SAVE (write to same URI) or SAVE AS

Notes
- All UI is constructed in Java code (no layout XML added).
- Saving writes UTF-8.
